The SKNCRE cosmetics brand content is served by GraphQL. Please [visit the playground](/api/graphql/skncre) to explore.

```sh
POST /api/graphql/skncre

```

## Query examples

### Get all products

Get a list of all products in the API with the relevant details. [See an example of this query in the playground](/api/graphql/skncre?query=query+MyQuery+%7B%0A++products+%7B%0A++++description%0A++++id%0A++++ingredients%0A++++name%0A++++price%0A++++shortDescription%0A++++slug%0A++++stock%0A++++images+%7B%0A++++++url%0A++++++alt%0A++++%7D%0A++%7D%0A%7D)

```graphql
query MyQuery {
  products {
    description
    id
    ingredients
    name
    price
    shortDescription

    stock
    images {
      url

    }
  }
}
```

<Tabs>
<TabItem label="Query">

```graphql
query MyQuery {
  product(id: "2") {
    description
    id
    ingredients
    name
    price

}
```
